# 网站功能规划

本文档以“当前已实现能力 + 后续建议”组织，避免与历史规划混淆。

## 1. 当前已实现模块

### 1.1 数据输入

- 上传本地 `.json` / `.txt` 文件
- 拖拽上传
- 输入在线书源 URL
- `Ctrl+V` 直接粘贴链接解析
- 批量文件上传
- 批量 URL 输入
- 在线 URL 安全校验

### 1.2 单个处理

- `只查重复`
- `全部校验`
- 去重前后统计
- 重复 URL 展示
- 格式失效统计
- 深度失效统计

### 1.3 深度校验

- 并发数选择：`1 / 4 / 8 / 16 / 32`
- 超时选择：`15 / 30 / 45 / 60`
- SSE 实时进度
- 当前处理 URL 和名称展示
- 预估剩余时间
- 取消校验
- 失败原因分组
- 失败分类展示
- 失败书源导出

### 1.4 类型过滤

- 正版
- 听书
- 漫画
- 影视

过滤依据包括：
- `bookSourceType`
- 名称/备注关键词
- 正版站点域名

### 1.5 批量处理

- 多文件合并处理
- 多 URL 合并处理
- 保留来源级统计：
  - `fileStats`
  - `urlStats`

### 1.6 搜索校验

- 仅搜索
- 仅发现
- 搜索+发现
- 搜索关键词选择与自定义输入
- 规则类型分布统计
- 搜索/发现失败分类

### 1.7 结果导出与展示

- 导出有效书源
- 导出失败书源
- 保留原分组
- 追加 `YYYY-MM-DD去重有效XXX条`

### 1.8 体验增强

- 亮色/暗色主题
- Toast 提示
- 本地记忆并发数、超时、主题

## 2. 当前产品结构

前端主模式：
- 单个处理
- 批量处理
- 搜索校验

后端主能力：
- 解析
- 去重
- 过滤
- 深度校验
- 搜索校验
- SSE 会话管理

## 3. 当前限制

- 无数据库，任务状态仅保存在进程内存
- 无用户系统
- 搜索规则中的复杂 JS 只支持部分解析
- 远程 URL 安全校验已实现基础防护，后续仍可继续加强

## 4. 后续建议

### 4.1 短期

- 补齐接口级自动化测试
- 收敛并拆分 `backend/app/api/sources.py`
- 清理前后端调试日志
- 继续加强 URL 安全校验与 SSRF 防护

### 4.2 中期

- 支持更多书源包装格式
- 增强搜索结果判定准确率
- 更细粒度的失败原因和修复建议
- 前端结果区块拆分，降低 `App.vue` 复杂度

### 4.3 长期

- 用户配置云端保存
- 历史任务记录
- 书源质量评分
- 可插拔规则校验扩展能力
